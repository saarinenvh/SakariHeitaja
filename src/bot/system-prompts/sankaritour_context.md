# SankariTour – Context

## Yleiskuva
SankariTour on Sankariheittäjien vuosittainen seuranmestaruuskilpailu.

- Järjestetty vuodesta 2017 lähtien
- Seuran tärkein kilpailukokonaisuus

## Terminologia
- SankariTour = Sankariheittäjien vuosittainen mestaruuskilpailu
- Osakilpailu = yksittäinen Tourin kilpailu
- Tasoitussarja = Metrix-ratingiin perustuva rinnakkainen kilpailusarja
- Raakatulos = pelaajan todellinen tulos ilman tasoitusta

## Rakenne
Tour koostuu useista osakilpailuista.

Tyypillinen rakenne:
- Avausosakilpailu toukokuussa
- Osakilpailu kesäkuussa
- Osakilpailu heinäkuussa
- Osakilpailu elokuussa
- Päätösosakilpailu syyskuussa

## Avajaiset ja finaali
- Useamman päivän tapahtumia
- Sisältävät majoituksen
- Sisältävät yhteisiä ruokailuja
- Ovat merkittävä osa seuran vuotuista toimintaa

## Kilpailun säännöt
- Perustuvat PDGA-sääntöihin
- Soveltaminen voi olla joustavaa tapahtuman luonteen mukaan

## Pisteytys
- Jokaisesta osakilpailusta saa sijoituksen mukaiset pisteet
- Kolme parasta osakilpailua lasketaan mukaan kokonaispisteisiin
- Eniten pisteitä kerännyt voittaa Tourin

## Sarjat
Kilpailussa on:
- Raakatuloksiin perustuva sarja
- Tasoituksellinen sarja

## Tasoitussarja
Tasoituksellinen SankariTour toimii pisteytyksen osalta samalla tavalla kuin normaali Tour:
- Jokaisesta osakilpailusta saa pisteitä sijoituksen mukaan
- Kolme parasta osakilpailua lasketaan kokonaispisteisiin
- Eniten pisteitä saanut voittaa

Erotus raakatulokseen:
- Jokaiselle pelaajalle lasketaan tasoitus ennen osakilpailua
- Tasoitus perustuu pelaajan Metrix-ratingiin
- Sama tasoitus koskee koko osakilpailua
- Tasoitus ei muutu kesken osakilpailun

Tasoitus määritetään ratingin perusteella:
- 900+ → 0
- 890–899 → -1
- 880–889 → -2
- 870–879 → -3
- 860–869 → -4
- 850–859 → -5
- 840–849 → -6
- 830–839 → -7
- 820–829 → -9
- 810–819 → -10
- 800–809 → -11
- 790–799 → -12
- 780–789 → -13
- 770–779 → -14
- 760–769 → -15
- 750–759 → -16
- 740–749 → -17
- 739 tai alle → -18

Tasoituksellinen tulos:
- Lasketaan lisäämällä tasoitus pelaajan raakatulokseen
- Lopullinen sijoitus voidaan määrittää tasoitettujen tulosten perusteella

## Palkinnot
- Kiertopokaali on kilpailun pääpalkinto
- Kilpailuissa voi olla myös erikoispalkintoja, kuten CTP-palkintoja

## Merkitys
SankariTour toimii:
- Seuran mestaruuden ratkaisevana kilpailuna
- Vuotuisena tapahtumakokonaisuutena
- Seuran yhteisenä kokoontumisena

## Kilpailun seuranta
- Kilpailut voidaan toteuttaa discgolfmetrixissä
- Sakke voi seurata kilpailua ja tallentaa tulokset
